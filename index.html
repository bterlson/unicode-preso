Level 1: Fonts

A font is a mapping of characters to glyphs.

It's a giant table of bitmaps or vectors
that represent characters.

A glyph is a visual representation of a character.

A font can contain many glyphs for the same character with multiple styles.

Ok, but how do you tell the font which glyph you want?

Level 2: Unicode & Code Points

Unicode is an international standard for
representing and working with text

It covers the majority of Earth's writing systems

It includes 283,506 characters as of Version 13

Unicode is also a giant table,
except it holds code points

A code point is an integer assigned to a character.

Often written like U+XXXX

Example Unicode Code Points

Code Point Name Block Glyph
U+0007 BELL Control Characters (none)
U+0041 LATIN CAPITAL LETTER A Basic Latin A

A grapheme is a combination of one or more code points that represent an atomic unit of a particular writing system

"Character" is ambiguous and the term should be avoided when talking about text.

Use code point when referring to an entry in the Unicode database

Use grapheme when referring to an atomic unit of a writing system (letter, punctuation, or emoji)

Use glyph when referring to a particular visual representation of a grapheme
